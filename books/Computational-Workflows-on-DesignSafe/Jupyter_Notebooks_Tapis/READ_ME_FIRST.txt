README: USING JUPYTER NOTEBOOKS IN DESIGNSAFE COMMUNITY DATA FOLDERS
====================================================================

IMPORTANT: THESE NOTEBOOKS ARE STORED IN A READ-ONLY LOCATION.

You CAN open and run these notebooks directly in this folder, but there
are some important limitations you should understand.


--------------------------------------------------------------------
READ-ONLY BEHAVIOR
--------------------------------------------------------------------

The notebooks in this folder are READ-ONLY.

This means:

- You CAN open and view the notebooks.
- You CAN execute the cells.
- You CAN temporarily edit the notebook while it is open.

HOWEVER:

- YOU CANNOT SAVE CHANGES TO THESE NOTEBOOKS.
- ANY EDITS YOU MAKE WILL BE LOST WHEN THE NOTEBOOK IS CLOSED.

In addition, notebooks in this folder should NOT attempt to write
output files to this directory.


--------------------------------------------------------------------
RECOMMENDED WORKFLOW
--------------------------------------------------------------------

IT IS STRONGLY RECOMMENDED THAT YOU COPY NOTEBOOKS TO YOUR OWN WORKSPACE
BEFORE WORKING WITH THEM.

This allows you to:
- Save changes
- Modify code
- Generate output files
- Safely experiment with the notebooks


--------------------------------------------------------------------
HOW TO COPY NOTEBOOKS IN JUPYTERHUB
--------------------------------------------------------------------

Copying notebooks is very easy using the JupyterHub file browser.

1. Locate the notebook you want to copy in the file navigation panel
   on the LEFT side of JupyterHub.

2. Select the notebook using your mouse.

3. RIGHT-CLICK on the selected notebook.

4. Click "Copy".

5. Navigate to the folder where you want to place the notebook
   (for example: your personal workspace folder).

6. RIGHT-CLICK inside that folder.

7. Click "Paste".

You can now open and edit the copied notebook normally.
